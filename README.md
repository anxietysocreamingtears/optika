# Global Project Portfolio

Статический портфолио-проект с выразительной типографикой, локальными шрифтами, плавными анимациями и минимальным JavaScript.

## Что внутри

- Одностраничный лендинг с блоками `hero`, `projects`, `about`, `services`, `process`, `quotes`, `contact`
- Локально подключённые шрифты `Cormorant Garamond` и `Manrope`
- Плавные `scroll reveal` анимации и лёгкий tilt-эффект для карточек проектов
- Адаптивная навигация для мобильных устройств

## Структура

- `index.html` — вся разметка страницы
- `assets/css/styles.css` — дизайн-система, сетки, адаптив и анимации
- `assets/js/main.js` — мобильное меню, reveal-анимации и tilt-эффект
- `assets/fonts/` — локальные шрифты
- `assets/images/favicon.svg` — иконка сайта

## Как открыть сайт

После пуша в `main` сайт публикуется через GitHub Pages.

Основная ссылка для этого репозитория:

- `https://anxietysocreamingtears.github.io/global_project/`

## Что проще всего заменить под себя

- Имя, тексты и контакты в `index.html`
- Палитру и визуальный характер через CSS-переменные в начале `assets/css/styles.css`
- Кейсы и описания в секции `#projects`
- Почту и GitHub в секции `#contact`

## Техническая заметка

Проект специально собран без тяжёлого стека: здесь достаточно HTML, CSS и небольшого количества JavaScript, чтобы сайт оставался лёгким и при этом выглядел дорого.
