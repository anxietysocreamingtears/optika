# Yarko Optika

Статический адаптивный сайт для салона оптики «Ярко оптика», собранный без backend и готовый к публикации через GitHub Pages.

## Что внутри

- `index.html` — основная структура лендинга
- `assets/css/styles.css` — дизайн-система, адаптивная сетка, анимации и карточный UI
- `assets/js/main.js` — мобильная навигация, reveal-анимации, tilt-эффект и фильтр каталога
- `assets/images/` — локальные SVG-ассеты: логотипы, hero-иллюстрация и продуктовые placeholder-изображения
- `.github/workflows/deploy-pages.yml` — автодеплой на GitHub Pages

## Особенности проекта

- Полностью статический стек: HTML, CSS, JavaScript
- Готовность к GitHub Pages без серверной части
- Mobile-first адаптивность
- Бело-оранжевая визуальная система с мягкими тенями и hover-анимациями
- Каталог с фильтрацией и поиском на чистом фронтенде

## GitHub Pages

После пуша в ветку `main` workflow публикует сайт на GitHub Pages.

Ожидаемый URL для этого репозитория:

- `https://anxietysocreamingtears.github.io/optika/`

## Что можно быстро заменить

- Тексты, контакты и адрес в `index.html`
- Палитру и визуальные акценты через CSS-переменные в `assets/css/styles.css`
- SVG-placeholder'ы в `assets/images/` на реальные фотографии товаров и интерьера

## Примечание по изображениям

В проект уже встроены брендовые графические ассеты по предоставленным логотипам. Для продуктовых карточек и hero-композиции пока использованы локальные SVG-placeholder'ы, которые можно безболезненно заменить на реальные фото.
